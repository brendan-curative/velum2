Alert Component

Alert components provide contextual feedback messages for typical user actions with a colored top border, icon, and close button.

Success Alert

This is a success alert

Your changes have been saved successfully.

Warning Alert

This is a warning alert

Please review your information before proceeding.

Critical/Error Alert

This is a fail alert

An error occurred while processing your request.

Usage

<div class="alert alert-success"> <div class="alert-border"></div> <div class="alert-content"> <div class="alert-main"> <div class="alert-icon"> <!-- SVG icon --> </div> <div class="alert-text"> <h4 class="alert-title">Alert Title</h4> <p class="alert-message">Alert message</p> </div> <button class="alert-close"> <!-- Close icon --> </button> </div> </div> </div>

Available Variants

  • .alert-success - Green success alert
  • .alert-warning - Orange warning alert
  • .alert-critical - Red error/critical alert
