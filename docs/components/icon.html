Icon Component

Icon component provides easy access to Heroicons using CSS background images. Icons are displayed as scalable background images with consistent sizing and positioning.

Usage Examples

Basic Icon

<span class="heroicon heroicon-user"></span>

Outline Icon

<span class="heroicon heroicon-outline heroicon-user"></span>

Small Icon (16px)

<span class="heroicon heroicon-16 heroicon-user"></span>

Medium Icon (20px)

<span class="heroicon heroicon-20 heroicon-user"></span>

With Accessibility

<span class="heroicon heroicon-user" aria-label="User profile"></span>

Available Icon Classes

Base Classes

  • .heroicon - Base class (24px, solid style)
  • .heroicon-outline - Use outline versions of 24px icons
  • .heroicon-16 - Use 16px mini icons
  • .heroicon-20 - Use 20px icons

Popular Icon Names

  • .heroicon-user - User profile
  • .heroicon-home - Home/dashboard
  • .heroicon-menu - Menu (bars-3)
  • .heroicon-x-mark - Close/cancel
  • .heroicon-chevron-down - Dropdown arrow
  • .heroicon-chevron-up - Collapse arrow
  • .heroicon-plus - Add/create
  • .heroicon-minus - Remove/subtract
  • .heroicon-pencil - Edit
  • .heroicon-trash - Delete
  • .heroicon-check - Confirm/success
  • .heroicon-heart - Favorite/like
  • .heroicon-star - Rating/bookmark
  • .heroicon-bell - Notifications
  • .heroicon-envelope - Email/messages
  • .heroicon-phone - Call/contact
  • .heroicon-magnifying-glass - Search
  • .heroicon-cog-6-tooth - Settings
  • .heroicon-information-circle - Info
  • .heroicon-exclamation-triangle - Warning
  • .heroicon-check-circle - Success status
  • .heroicon-x-circle - Error status

Technical Details

  • Implementation: CSS background images
  • Source: Heroicons library (1,288+ icons available)
  • Accessibility: Include aria-label for screen readers
  • Performance: Icons load on-demand as background images
  • Customizable: Size and color can be adjusted with CSS
  • Semantic: Use <span> or other appropriate elements
